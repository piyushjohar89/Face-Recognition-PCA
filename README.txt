INSTRUCTIONS TO RUN CODE

1.Extract the zip file and copy the contents into the suitable MATLAB folder of your choice.

2.In the function extract_face.m and testing.m replace the path with the path of your folder containing the training images.

3. To run face_recognition.m, type the following command in the command window:
face_recognition

4. Choose existing file from the folder where training files are - traindata. 

5. If you wish to capture your image, it will save in image folder. Again chose this file from image folder to see application results.
